[ تبويب "تشفير / فك تشفير" ]

  التشفير:
  1. اضغط منطقة الصورة أو اسحب ملف صورة عليها
  2. اكتب كلمة المرور (اضغط 👁 للإظهار)
  3. اضغط "تشفير ملف داخل الصورة"
  4. اختر الملف السري المراد إخفاؤه
  5. اختر مكان حفظ الصورة الناتجة

  فك التشفير:
  1. اختر الصورة المشفرة
  2. اكتب كلمة المرور الصحيحة
  3. اضغط "فك التشفير واستخراج الملف"
  4. الملف يُحفظ تلقائياً في مجلد "Extracted_Files"
     بجانب الصورة المشفرة باسمه الأصلي
